# xAI-CPAVAE_for_age_progression_and_regression
xAI-CPAVAE: Enhancing Deep Learning Generative Networks with Explainable AI:  This research-driven project focuses on developing a Conditional Perceptual Adversarial Variational Auto-encoder (CPAVAE) integrated with explainable AI techniques. Our goal is to improve the interpretability and robustness of deep learning-based generative networks.
